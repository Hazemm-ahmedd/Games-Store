# 🎮 GameVault - Premier Gaming Store

A modern, highly responsive, and premium gaming e-commerce front-end interface featuring a sleek cyber-dark theme. This project simulates a real-world digital game store experience, complete with an interactive shopping cart system, dynamic catalog search, and a mock secure payment gateway.

## 🚀 Live Demo
[View Live Demo](https://hazemm-ahmedd.github.io/Games-Store/)
## ✨ Key Features
- **Premium Cyberpunk Design:** Crafted with an immersive dark layout, glowing neon accents, and smooth cinematic transitions.
- **Dynamic Shopping Cart System:** Users can add/remove games from the vault bag in real-time. The cart automatically updates the item counter and calculates the precise total bill.
- **Instant Catalog Engine:** A lightweight search bar filter that allows users to seamlessly scan and find game titles, DLCs, or bundles instantly without page reloads.
- **Secure Mock Checkout:** A completely simulated payment modal supporting multiple virtual gateways (Credit Card, PayPal, Crypto) with form validation mechanisms.
- **Interactive System Toasts:** Displays immediate non-intrusive feedback notifications when items are successfully appended to the cart.
- **Fully Responsive Dictation:** Optimized across all device structures including desktops, tablets, and smartphones.

## 🛠️ Tech Stack
- **Structure:** HTML5 (Semantic Elements)
- **Styling:** CSS3 (Custom Variables, CSS Grid, Flexbox, Dynamic Media Queries)
- **Interactions:** Vanilla JavaScript (ES6 Document Object Model API, Event Delegation)
- **Typography & Icons:** FontAwesome v6.4 & Google Fonts (Outfit Pack)
